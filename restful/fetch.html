fetch(input[,init])

概念

  • fetch向服务器发送请求，返回一个promise对象，这个promise对象resolve请求的response
  • 返回的promise对象只会reject network error；
  • promise的then()方法必须检查http error例如404，即检查Response.status

参数

  • input：URL/request object
  • init：一个对象，包含request参数：例如method，headers，body等等

fetch api

Headers对象
Headers API用来处理request或response的请求头对象
新建一个headers对象的实例，供request和response对象使用
.get()/.append/.delete/.has/.set/.keys/.values/.entries
Request对象
Request对象实例，作为fetch()的参数，定义请求的内容和细节
request属性都是只读，读取request信息。
Response对象
Response对象实例，作为fetch()返回promise的resovle参数，是请求得到的返回结果
response.status/.ok/.header/
